== RestFixture
A FitNesse fixture for testing REST API.
The main purpose of this fixture is to allow developers or product
owners to write tests for REST APIs in a simple and effective way.
An overview of the features of this fixture is available here
http://smartrics.blogspot.com/2008/08/get-fitnesse-with-some-rest.html
